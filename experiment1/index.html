Experiment 1 - Getting Started

Description

This generator is remixed from my professor's template. It is simply made by HTML, CSS, and JavaScript.
Every time you click the "REROLL" button, it will randomly generate some different words. It is not a story or something, it is just some random shit for fun.

Technical

We need to create an array that has a specific name with different words. Then It will based on a replacer() function to pick the random words. When the replacer() function catches any replaceable name, then it will randomly pick a word from that name array.
